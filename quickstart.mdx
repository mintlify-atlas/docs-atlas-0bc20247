By the end of this guide you will have MalwareUI running locally — a dark 800×600 window with a countdown timer, ransom message panel, and status display. The entire setup takes less than five minutes and requires only Python and pip.

<Warning>
  MalwareUI is a demonstration tool only. It performs no encryption, makes no system changes, and transmits no data. Use it solely for education, security awareness training, or UI prototyping.
</Warning>

<Info>
  Python 3.8 or later is required. Run `python --version` to confirm before proceeding.
</Info>

## Installation

<Steps>
  <Step title="Clone the repository">
    Download the source code to your local machine:

    ```bash
    git clone https://github.com/Kars1996/MalwareUI.git
    cd MalwareUI
    ```
  </Step>

  <Step title="Create a virtual environment">
    <Tip>
      Running in a virtual environment keeps MalwareUI's dependencies isolated from your system Python, preventing version conflicts with other projects.
    </Tip>

    <Tabs>
      <Tab title="macOS / Linux">
        ```bash
        python3 -m venv .venv
        source .venv/bin/activate
        ```
      </Tab>
      <Tab title="Windows">
        ```bash
        python -m venv .venv
        .venv\Scripts\activate
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Install requirements">
    Install the two pinned dependencies — `customtkinter==5.2.1` and `pillow==10.2.0`:

    ```bash
    pip install -r requirements.txt
    ```
  </Step>

  <Step title="Run the application">
    Launch the demo window:

    ```bash
    python src/main.py
    ```
  </Step>
</Steps>

## What to expect

When the application starts you will see an 800×600 dark window (`fg_color="#1a1a1a"`) with the following layout:

| Region | Content |
|---|---|
| Header | "SYSTEM COMPROMISED" in red (`#ff0000`), 32 pt Terminal bold |
| Content frame | CountdownTimer (2-hour countdown), MessageDisplay, StatusPanel |
| Footer | Static BTC address in green (`#00ff00`), Consolas 12 pt |

The three components — `CountdownTimer`, `MessageDisplay`, and `StatusPanel` — are stacked vertically inside a dark content frame (`fg_color="#2d2d2d"`). The countdown begins immediately on launch and counts down from two hours.

To close the application, press the standard window close button or `Ctrl+C` in the terminal.
