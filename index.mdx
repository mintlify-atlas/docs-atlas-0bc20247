MalwareUI is a Python desktop application built with CustomTkinter that demonstrates the visual design of a ransomware interface. It features an animated countdown timer, a threatening message panel, and a simulated file encryption status display — all purely cosmetic with no actual system modifications.

<Warning>
  This project is a demonstration UI only. No actual encryption, file modification, or system changes are made. It is intended for educational purposes, UI/UX research, and cybersecurity awareness demonstrations.
</Warning>

## How it works

MalwareUI is composed of three independent CustomTkinter components assembled inside a main application window. Each component handles its own rendering and animation logic.

<Steps>
  <Step title="Install dependencies">
    Create a virtual environment and install `customtkinter` and `pillow` via pip.
  </Step>
  <Step title="Run the application">
    Execute `python src/main.py` to launch the dark-themed UI window.
  </Step>
  <Step title="Watch the demo">
    The countdown timer starts, the status panel animates through simulated file encryption, and the threatening message is displayed — all in real time.
  </Step>
  <Step title="Customize">
    Modify component parameters such as `hours`, message text, colors, and progress timing to tailor the demo to your needs.
  </Step>
</Steps>
